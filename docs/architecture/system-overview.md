# DRF Project Flow Architecture
```
┌─────────────────────────────────────┐
│        클라이언트 애플리케이션          │
│    (React, Vue, Mobile App 등)      │
└─────────────────┬───────────────────┘
                  │ HTTP/JSON API
┌─────────────────▼───────────────────┐
│          API Gateway Layer          │
│      (Authentication, CORS 등)      │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│         ViewSet/APIView Layer       │
│        (API 엔드포인트 처리)          │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│        Serializer Layer            │
│       (데이터 직렬화/역직렬화)         │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│         Business Logic Layer        │
│           (Service Layer)           │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│            Model Layer              │
│         (데이터 모델 & ORM)           │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│           Database Layer            │
│        (PostgreSQL)                 │
└─────────────────────────────────────┘
```
